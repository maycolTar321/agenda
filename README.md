# Z-Core: Life Optimizer 🌌

¡Bienvenido al sistema de productividad definitivo inspirado en la tecnología de la **Capsule Corp** y el **Ultra Instinto** de Dragon Ball Super!

## ✨ Características Principales
- **Agenda Z:** Gestión inteligente de tu tiempo con estética Scouter.
- **Misiones Diarias:** Sistema de tareas con rangos (S, A, B) que aumentan tu Poder de Pelea.
- **Suministros:** Lista de compras integrada.
- **Cámara de Gravedad:** Temporizador de enfoque (Pomodoro) con estética de entrenamiento de Vegeta.
- **Nivel de Poder:** Visualiza tu progreso diario. Al llegar al 100%, ¡desbloqueas el modo Ultra Instinto!
- **PWA Ready:** Instalable en Android y disponible offline.

## 🚀 Instalación en GitHub
1. Sube todos estos archivos a un nuevo repositorio en GitHub.
2. Ve a `Settings` -> `Pages`.
3. Selecciona la rama `main` y la carpeta `/root`.
4. ¡Listo! Tu app estará viva en `https://tu-usuario.github.io/tu-repo/`.

## 📱 Conversión a Android (.apk)
Para tener Z-Core como una app nativa en tu celular:
1. Usa [Bubblewrap](https://github.com/GoogleChromeLabs/bubblewrap) o una herramienta online como [Web2App](https://web2app.com/).
2. Introduce el link de tu GitHub Pages.
3. Descarga el archivo `.apk` e instálalo en tu dispositivo.

## 🛠️ Tecnologías
- **Frontend:** HTML5, CSS3 (Glassmorphism, Animations), JS ES6.
- **Persistencia:** LocalStorage.
- **PWA:** Service Workers y Manifest.
